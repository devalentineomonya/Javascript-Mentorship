Understanding JavaScript Variables

Variables in JavaScript are containers for storing data values. Variables can hold different types of data, such as numbers, strings, and objects.

Linking JavaScript Files to HTML

There are two primary ways to link JavaScript files to an HTML file:

  • In the head: Link the JavaScript file inside the <head> section. This runs the script before the content loads.
    • <script src="variables.js"></script>
  • Before the closing body tag: Placing the script link before </body> ensures the HTML content loads first.
    • <script src="exercise.js"></script>
